Parks Report

Currently a work in progress, Parks Report is a Rails 5 app that displays visitor data from all 59 national parks. Data visualization through Chartkick and Google Charts.

Sized for Slack

Image resizing app written in Go. Enables users to upload a JPEG and download a resized image that meets the requirements of Slack's custom emoji.

Foster Network

Built in the Fall of 2015 while attending Launch Academy. Foster Network is a Rails app that allows users to import a group of animals through the PetFinder API to find them foster homes.
